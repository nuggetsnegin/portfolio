About

Hi there, I'm Negin [nə'giːn] and I'm a front-end developer based in Toronto, Canada! I got my start in web-development back in the early 2000s when I was styling my amazing Neopets' profile page and creating Sailor Moon fan pages in Geocities. I'm passionate about creating beautiful, responsive and accessible websites.

I have a Bachelor of Applied Computer science from Dalhousie University and a Certificate of Web Development Immersive from Juno College of Technology.

When I'm not coding by moonlight - I'm drawing, running, or playing N64 games (all at the same time!)

Oh, and I have 3 cats.ヽ(=^･ω･^=)丿

Feline Feeling

HTML5 CSS3 JavaScript JQuery

A quirky game where you have to find the cat's purfurred petting spot in under 30 seconds.

Features 6 randomized petting spots and over 9 different randomized cats (including my personal favorite - Garfield!)

Art assets made by yours truly and music is from Wario Land 3 and Kirby SuperStar.

Wishy Washy

HTML5 CSS3 JavaScript React Firebase

A wholesome wishing board! Users can anonymously input a wish and their wish bubble will appear in the Wishy Washy ocean.

Features wish support where users can heart other wishes to show their support. 9 wishes will randomly be rendered whenever the user enters a wish or refreshes the page and the most recent wish will always be displayed.

Furiend Finder

HTML5 CSS3 API JQuery JavaScript Purr Programming

A pet adoption site created with the amazing Juan Acaiturri-Villa featuring 4 different APIs to find adoptable rescue dogs and cats within a user's location.

Users can filter pets by location and age through the PetFinder and MapQuest API and receive breed facts from the Cat and Dog API (yes, that's the API name!) Art assets made by yours truly.
